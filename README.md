# Lid-CRM - E-Commerce Platform with Admin Panel

A complete e-commerce solution for dropshipping with a powerful admin panel.

## Features

### Customer Features
- Product catalog with search and filters
- Shopping cart management
- Secure checkout
- Order tracking
- User account management
- Product reviews and ratings

### Admin Features
- Complete product management (CRUD)
- Order management and tracking
- User management
- Sales analytics and reports
- Inventory management
- Discount and coupon management
- Website content management
- Admin user role management
- Revenue tracking

## Tech Stack

- **Frontend**: React 18+, Redux, Tailwind CSS
- **Admin Panel**: React, Recharts for analytics
- **Backend**: Node.js, Express, JWT Auth
- **Database**: MongoDB
- **Deployment**: Docker, ready for cloud deployment

## Project Structure

```
Lid-CRM/
├── backend/          # Node.js API server
├── frontend/         # Customer-facing React app
├── admin-panel/      # Admin dashboard
├── docker-compose.yml
├── .env.example
└── README.md
```

## Quick Start

1. Clone the repository
2. Copy `.env.example` to `.env` and configure
3. Run `docker-compose up` to start all services
4. Frontend: http://localhost:3000
5. Admin: http://localhost:3001
6. API: http://localhost:5000
